 - - Comandos basicos
git init - cria um repositorio git
git clone - clona um repositorio pro pc
cd - navegar pelos diretórios
pull (puxar) - traz do repositorio pra minha maquina (contrario do git push)
push (empurrar) - traz da minha maquina pro repositorio (contrario do git pull)
merge (juntar as branches) - combinar o que foi feito em uma no que foi feito na outra (utilizar no fim, quando terminei 
de usar a branch)
pwd (printing working directory/imprimir o diretorio de trabalho)- em que diretorio estou atualmente

___

Passo 1 - Comando cd "path(caminho)-do-diretorio" navega pelos diretorios
Passo 2 - git clone ((HTTPS)> link do repositorio do navegador)
Passo 3 - git status (pra verificar se tem alteracoes na branch)
Passo 4 - git add "nome-do-arquivo" ou .(se for add tudo do diretorio) (pra adicionar arquivos do diretorio)
Passo 5 - git status (Desnecessario segundo o leo)
Passo 6 - git commit -m "mensagem/comentario/observacoes sobre oq foi alterado"
Passo 7 - git push origin "nome-da-branch" (subir o codigo do meu computador pra branch do repositorio)
